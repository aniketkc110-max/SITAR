# 🎵 Sitar Tuner

A beautiful, free, open-source sitar tuner that works in any web browser — on laptop or mobile phone.

## Features

- 🎙 **Live microphone pitch detection** — just play your string and it detects the note
- 🪕 **6 common sitar tunings** included:
  - Standard Kharaj Pa (C#-G#)
  - Kharaj Ma (C-G)
  - Vilayat Khan Style (D-A)
  - Ravi Shankar Style (E-B / C#)
  - Ustad Imrat Khan Style (F-C)
  - Beginner Practice (A-E)
- 🎯 **Visual needle meter** — shows cents sharp/flat
- ✅ **String-by-string feedback** — each string glows green when in tune
- 📱 **Works on phone and laptop** — no app install needed

## How to Use

1. Open `index.html` in your browser (Chrome recommended)
2. Select your tuning preset
3. Tap **Start Tuning** and allow microphone access
4. Play a string on your sitar — the tuner shows the detected note and how sharp/flat you are
5. Tune until the needle is centered and the string glows green ✅

## How to Put on GitHub

1. Go to [github.com](https://github.com) and sign in (or create a free account)
2. Click the **+** button → **New repository**
3. Name it `sitar-tuner`, set it to **Public**
4. Click **Create repository**
5. Upload `index.html` and this `README.md`
6. Go to **Settings → Pages → Source: main branch** to publish it live for free!

## Tech

- Pure HTML, CSS, JavaScript — no frameworks, no installs
- Uses **Web Audio API** for real-time microphone pitch detection
- Auto-correlate algorithm for accurate frequency detection
